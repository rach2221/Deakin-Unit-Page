SIT223-SIT753 - Professional Practice in IT

Unit Overview

Content

As students, this unit introduces key industry tools, professional practices, and teamwork skills required in IT environments. It focuses on how technologies integrate, while also developing ethical awareness, communication skills, and problem-solving abilities. This allows students to clearly understand expectations and prepares them for real-world IT careers.

What You Will Learn

  • Use of industry-standard tools and technologies
  • Professional communication and teamwork skills
  • Ethical responsibilities in IT practice
  • Problem-solving and critical thinking
  • Collaboration using version control systems (e.g., Git and GitHub)

Hurdle Requirements

To successfully pass this unit, students must complete all required tasks and demonstrate consistent engagement throughout the trimester.

  • Complete all portfolio tasks and assessments
  • Actively participate in group work and team activities
  • Demonstrate practical understanding of IT tools and concepts
  • Meet all deadlines for submissions
  • Show evidence of collaboration using Git and GitHub

Important Notes

  • Failure to meet hurdle requirements may result in failing the unit
  • All assessments must be attempted
  • Academic integrity must be maintained at all times
